# Testing Methodology

## Local Gates

Use `npm test` for smoke coverage and `npm run validate:release` for release readiness.

## Behavioral Coverage Targets

- Routing fixture tasks.
- Context pack snapshot checks.
- Package content checks.
- Skill metadata and required-section checks.
- Reference quality checks.

## AI-Assisted Benchmarks

AI benchmark results are supporting evidence only. They do not replace real users, CI, or maintainer review.
